# R&B Power — Site Appraiser

Internal tool for appraising energy site value based on power capacity, acreage, and project parameters.

## Stack

- React + TypeScript
- Vite
- Tailwind CSS v4
- Firebase (Firestore for site data persistence)

## Development

```bash
npm install
npm run dev
```

## Deployment

Pushes to `main` auto-deploy to GitHub Pages via GitHub Actions.

Live at: [jbb987.github.io/randb-site-valuator](https://jbb987.github.io/randb-site-valuator/)
